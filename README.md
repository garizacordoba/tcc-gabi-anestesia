# 🩺 Sistema de Orientação Pré-Anestésica Digital

Este Web App foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)** para a Residência de Anestesiologia.

## 🎯 Objetivo Principal
Facilitar o acesso às orientações pré-operatórias para pacientes que **residem em cidades distantes**, eliminando a necessidade de deslocamentos apenas para orientações básicas de preparo. O sistema visa reduzir o cancelamento de cirurgias e aumentar a segurança do paciente através de orientações digitais claras e precisas.

## ✨ O que o App faz:
- **Triagem de Risco:** Avalia automaticamente critérios de Via Aérea Difícil (VAD) e Hipertermia Maligna.
- **Protocolo de Medicamentos (V45):** Analisa a lista de remédios do paciente e gera instruções automáticas de suspensão ou manutenção (ex: Anticoagulantes, GLP-1, Insulinas).
- **Relatório PDF:** Gera um documento formatado com instruções de jejum e preparo para o paciente imprimir.
- **Área Médica:** Resumo técnico para o anestesista com alertas de segurança.

## 🚀 Tecnologias
